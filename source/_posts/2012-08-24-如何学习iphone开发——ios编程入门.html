末学浅薄，2011年10月才开始接触iOS开发，走过一些弯路，目前正全职从事iOS开发工作。在此分享一下学习iOS开发的过程。


初学请先看，

iOS Programming - The Big Nerd Ranch Guide - 3rd Edition

网上有电子书。

第三版最近出的，主要是针对iOS 5, Xcode 4.3加了一些新东西。

末学只看过第二版， 觉得当中的例子很适合边学习边编程这种方式，这本书的例子覆盖了大多常用的API，但不算深入。

把书中的例子实现并且部署以后，会对iOS开发有更具象的认识。
想写出格式更专业的代码，可以看看google的objc风格指南，

http://google-styleguide.googlecode.com/svn/trunk/objcguide.xml

http://zh-google-styleguide.readthedocs.org/en/latest/google-objc-styleguide/
开发自己的应用，

看书的过程中，请想个点子，然后动手去实现，这很重要，如果经济允许，并打算以后从事这方面的工作，最好花点钱讲你的应用部署到App store上。因为这个部署过程也较复杂。
编程的过程中，多看，

Apple 官方文档

Apple的文档是会常需要拿出来参考，反复读的资料，无论是泛读还是细读，在不同阶段，针对不同的问题，都会有需要。因为这里最详尽。

比如下面这些，当然还有更多，都可以去iOS Developer Library里找，你会发现很多文档都值得看看。

View Programming Guide

http://developer.apple.com/library/ios/#documentation/windowsviews/conceptual/viewpg_iphoneos/Introduction/Introduction.html

View Controller Programming Guide for iOS

http://developer.apple.com/library/ios/#featuredarticles/ViewControllerPGforiPhoneOS/Introduction/Introduction.html#//apple_ref/doc/uid/TP40007457

Table View Programming Guide for iOS

http://developer.apple.com/library/ios/#documentation/UserExperience/Conceptual/TableView_iPhone/AboutTableViewsiPhone/AboutTableViewsiPhone.html
许多专题，特别是iOS5/6的最新功能，WWDC视频，keynotes 是最前沿的参考，

https://developer.apple.com/videos/wwdc/2012/

（这里只贴了2012年的，以前的也同样值得看）

可以有选择有针对的看，比如你想具体了解iOS 6的Passkit编程细节，想了解最新的auto layout技术，都可以在这里找到。
当写了1，2个月实际应用以后，想深入理解Cocoa框架，并写出高质量，良好结构的Objetive C代码，要看

Cocoa Design Patterns

网上有电子书。

这是一本面向整个Cocoa框架的书，虽然主要是针对Mac编程，但OS X Library 和 iOS Library非常相似，设计模式更是一致。所以即使是只想做iOS开发，对Mac开发没有打算的人，也应该去看。

其他

Programming Objective-C

iOS x in Action

Xcode 4 iOS Development Beginner's Guide
