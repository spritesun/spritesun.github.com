Basically, I want to use java 5 as the my default jvm on Mac OS X, Version 10.6.3. But the default java version is 1.6.

(This blog just want to share with you that how to get java 5 or other java low then 1.6 be installed in mac os x snow leopard.)

>>> What jvm in your machine?

You could find various version of Java here,

/System/Library/Frameworks/JavaVM.framework/Versions

if you do a list, you may got that,

lrwxr-xr-x   1 root       wheel   10 Jun  1 23:21 1.4 -> CurrentJDK

lrwxr-xr-x   1 root       wheel   10 Jun  1 23:21 1.4.2 -> CurrentJDK

lrwxr-xr-x   1 root       wheel    5 Jun  7 01:11 1.5 -> CurrentJDK

lrwxr-xr-x   1 root       wheel   14 Jun  7 01:10 1.5.0 -> CurrentJDK

lrwxr-xr-x   1 root       wheel    5 Jun  1 23:21 1.6 -> 1.6.0

drwxr-xr-x  10 root       wheel  340 Jun  7 01:16 1.6.0

lrwxr-xr-x   1 root       wheel    3 Jun  7 01:16 CurrentJDK -> 1.6

>>> It may cheat you。

You will find that even there seems offer java from version 1.3 to 1.6, but all the folder finally link to the same place "1.6.0".

So actually, you only has java 1.6 on your machine, all the other version are "cheat" folder, they are “1.6.0" inside.

>>> Just remove the cheat link

What you need to do is that download java 5 package, link the CurrentJDK to that, and then change "Java Preferences".

Please refer to the following link, to get more details about java 5 on OS X 10.6

http://wiki.oneswarm.org/index.php/OS_X_10.6_Snow_Leopard
